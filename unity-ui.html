캔버스

캔버스 생성하기

  1. Hierarchy | 마우스 오른쪽 클릭 | UI | Text 메뉴를 선택합니다.

  2. Canvas, Text, EventSystem 이 생성됩니다.

  3. Scene | 2D 를 선택하여, 평면으로 보여지도록 변경합니다.

  4. Hierarchy 윈도우에서 Canvas 를 선택하고, Scene 윈도우쪽으로 마우스를 이동시키고 F 를 누릅니다.

  5. 캔버스가 포커스 됩니다.

    중간에 Text 가 있지만, 작아서 잘 보이지 않습니다.

EventSystem 은 클릭과 같은 이벤트를 탐지하는 오브젝트로 신경쓰지 않으셔도 됩니다.

유아이 요소 추가하기

텍스트 추가

  1. 추가된 Text 의 Width와 Height를 (700, 400)으로 변경합니다.

  2. Text를 Hello!!로 변경하고, Font Size를 150으로 변경합니다.

  3. 캔버스에 텍스트가 출력됩니다.

  4. 이동 기즈모를 사용하여 가장자리로 이동시킵니다.

이미지 추가

  1. 인터넷에서 적당한 이미지를 찾아 Project에 추가해 줍니다.

  2. 유아이에 사용하는 이미지는 Texture Type을 Sprite (2D and UI)로 변경해 줍니다.

    Apply 버튼을 클릭하여 변경사항을 적용시켜줍니다.

  3. 캔버스를 마우스 오른쪽 버튼으로 클릭한 뒤, UI | Image 메뉴를 선택하여 이미지를 추가해 줍니다.

  4. Inspector | Image | Source Image에 추가한 이미지를 드래그하여 적용시킵니다.

  5. Set Native Size 버튼을 클릭하면 이미지가 원본 사이즈로 확대 됩니다.

  6. 캔버스에 이미지가 추가됩니다.

  7. 이미지도 가장자리로 위치를 변경합니다.

버튼 추가

  1. Canvas | 마우스 우클릭 | UI | Button 메뉴를 선택하여, 버튼을 추가합니다.

  2. 버튼의 Width, Height를 (700, 400)으로 변경합니다.

  3. Button 하위의 Text를 선택합니다.

  4. 버튼 Text의 Font Size를 200으로 변경합니다.

  5. 버튼의 위치도 변경해 줍니다.

유아이 스크립트 연결하기

유아이 스크립트 추가

  1. import { ZepetoScriptBehaviour } from 'ZEPETO.Script' export default class UiManager extends ZepetoScriptBehaviour { Start() { } }

    UiManager 이름의 스크립트를 하나 추가합니다.

  2. import { Button, Text } from 'UnityEngine.UI' import { ZepetoScriptBehaviour } from 'ZEPETO.Script' export default class UiManager extends ZepetoScriptBehaviour { public text: Text; public button: Button; Start() { } }

    Text, Button 타입의 프로퍼티 text, button을 선언해 줍니다.

  3. Canvas에 UiManager 스크립트를 드래그하여 추가해 줍니다.

  4. Canvas 하위의 Text와 Button을 각각 드래그하여 프로퍼티를 할당해 줍니다.

유아이 조작하기

  1. import { Button, Text } from 'UnityEngine.UI' import { ZepetoScriptBehaviour } from 'ZEPETO.Script' export default class UiManager extends ZepetoScriptBehaviour { public text: Text; public button: Button; Start() { this.text.text = "count = 0"; } }

    Text는 text라는 이름의 프로퍼티를 가지고 있으며, 이 값은 string 타입입니다.

    이 text 값을 변경해주면, 텍스트의 문구를 변경할 수 있습니다.

  2. 실행 시, 텍스트의 문구가 변경된 것을 확인할 수 있습니다.

  3. import { Button, Text } from 'UnityEngine.UI' import { ZepetoScriptBehaviour } from 'ZEPETO.Script' export default class UiManager extends ZepetoScriptBehaviour { public text: Text; public button: Button; Start() { this.text.text = "count = 0"; this.button.onClick.AddListener(() => { }); } }

    버튼 클릭 시, 로직이 동작하도록 하기 위해 Button.onClick.AddListener를 사용합니다.

    AddListener 내부에는 () => { }과 같은 형태로 추가되며, 중괄호 내부에 버튼 클릭 시 동작할 로직을 작성합니다.

  4. import { Button, Text } from 'UnityEngine.UI' import { ZepetoScriptBehaviour } from 'ZEPETO.Script' export default class UiManager extends ZepetoScriptBehaviour { public text: Text; public button: Button; private count: number = 0; Start() { this.text.text = "count = 0"; this.button.onClick.AddListener(() => { this.count = this.count + 1; this.text.text = `count = ${this.count}`; }); } }

    count 이름의 프로퍼티를 추가하고, 버튼 클릭 시마다 1 증가시키고 현재 값을 텍스트에 출력시키도록 변경합니다.

  5. 버튼을 클릭할 때마다 1씩 증가하면서 텍스트에 count가 출력됩니다.
