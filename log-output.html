스크립트 파일 열기

스크립트 파일을 열어 코드 에디터를 실행 시킵니다.

코드 에디터 실행

  1. Project | Assets | SampleScript 파일을 더블 클릭 합니다.

  2. VSCode(Visual Studio Code) 를 통해 해당 스크립트 코드가 열립니다.

  3. Start 와 Update 의 코드 내용을 제거합니다.

    import { ZepetoScriptBehaviour } from 'ZEPETO.Script' export default class SampleScript extends ZepetoScriptBehaviour { Start() { } Update() { } }

로그 출력

로그 출력 코드를 추가하여 간단한 로그를 출력합니다.

기본 로그 출력

  1. Start 부분에 코드를 추가합니다.

    import { ZepetoScriptBehaviour } from 'ZEPETO.Script' export default class SampleScript extends ZepetoScriptBehaviour { Start() { console.log("log"); console.warn("warn"); console.error("error"); } Update() { } }

  2. Play 버튼을 클릭하여, 재생합니다.

  3. Window | General | Console 메뉴를 통해 콘솔 윈도우를 활성화 합니다.

  4. 콘솔에 로그가 출력되는 것을 확인합니다.

    코드의 진행 중간마다 로그를 추가하면, 진행 상황을 파악하거나 이슈를 확인하는데 도움이 됩니다.
